# 행운의서동이 — Vercel 무료 배포 가이드

## 준비물
- 구글 계정 (또는 GitHub 계정)
- 이 폴더의 파일들

## 배포 방법 (10분 소요)

### 1단계: GitHub 가입 (이미 있으면 건너뛰기)
1. https://github.com 접속
2. "Sign up" 클릭
3. 이메일, 비밀번호 입력하고 가입 (무료)

### 2단계: GitHub에 코드 올리기
1. GitHub 로그인 후, 오른쪽 위 "+" 버튼 → "New repository" 클릭
2. Repository name: `haengunyi` 입력
3. "Public" 선택
4. "Create repository" 클릭
5. 만들어진 페이지에서 "uploading an existing file" 링크 클릭
6. 이 폴더 안의 모든 파일/폴더를 드래그해서 업로드:
   - `api/` 폴더 (bundle.js 포함)
   - `public/` 폴더 (index.html, style.css, app.js 포함)
   - `package.json`
   - `vercel.json`
7. "Commit changes" 클릭

### 3단계: Vercel 가입 + 배포
1. https://vercel.com 접속
2. "Sign Up" → "Continue with GitHub" 클릭
3. GitHub 계정 연동 허용
4. 대시보드에서 "Add New..." → "Project" 클릭
5. 방금 만든 `haengunyi` 저장소 선택 → "Import"
6. 설정은 아무것도 건드리지 말고 바로 "Deploy" 클릭
7. 1~2분 기다리면 배포 완료!

### 4단계: 완성!
- 배포가 끝나면 `https://haengunyi.vercel.app` 같은 주소가 생김
- 이 주소를 공유하면 누구나 언제든 접속 가능
- 서버가 꺼질 걱정 없음 (Vercel이 자동 관리)

## 파일 구조
```
haengunyi-vercel/
├── api/
│   └── bundle.js        ← 서버리스 함수 (주식 데이터 수집)
├── public/
│   ├── index.html        ← 메인 페이지
│   ├── style.css         ← 다크 테마 스타일
│   └── app.js            ← 분석 엔진 + UI
├── package.json
├── vercel.json           ← 라우팅 설정
└── 배포가이드.md          ← 이 파일
```

## 참고
- Vercel Hobby 플랜: 완전 무료, 카드 등록 불필요
- 월 100GB 대역폭, 서버리스 함수 무제한 (일반 사용 충분)
- 코드 수정하고 싶으면 GitHub에서 파일 수정 → Vercel이 자동 재배포
