TSS 분석 방법론

행운의서동이는 어떻게 종목을 분석하는가

TSS란?

TSS(Total Stock Score)는 행운의서동이가 개발한 종합 주식 평가 점수입니다.

"이 종목 지금 사도 될까?" — 이 단순한 질문에
감이 아닌 숫자로 답하기 위해 만들었습니다.

하나의 종목을 5가지 관점에서 분석하고, 시장 국면에 따라 가중치를 자동 조절한 뒤, 100점 만점의 점수로 환산합니다. 여기에 5가지 안전 필터를 추가로 통과시켜, 숫자만 좋아 보이는 함정 종목을 걸러냅니다.

PROCESS

8단계 분석 프로세스

모든 종목은 아래 8단계를 순서대로 거칩니다. 단 한 단계도 건너뛰지 않습니다.

1
시장 데이터 수집
현재가, 전일대비, 거래량, PER, PBR, EPS, 52주 고/저, 목표가, 1년치 일봉 차트를 실시간으로 수집합니다.
2
시장 국면 진단
5일/20일/60일 이동평균선 배열과 현재가 위치를 비교해 상승 추세, 하락 추세, 반등 국면, 횡보 국면 4가지 중 하나로 진단합니다.
3
국면별 가중치 조절
상승장에서는 모멘텀과 기술적 지표에 더 큰 비중을, 하락장에서는 가치와 안전성에 더 큰 비중을 둡니다. 시장 상황에 맞게 판단 기준이 자동으로 바뀝니다.
4
5대 카테고리 채점
가치(Value), 성장(Growth), 모멘텀(Momentum), 수급(Supply), 기술(Technical) — 각 항목을 0~100점으로 독립 채점합니다.
5
TSS 산출
5개 카테고리 점수에 국면별 가중치를 곱해 가중 평균을 계산합니다. 이것이 TSS(0~100)입니다.
6
안전 필터 점검
TSS 점수가 높아도 위험한 종목이 있습니다. 5가지 안전 필터로 급락, 이상 거래량, 과대평가, 추세 이탈, 과도한 변동성을 검사합니다.
7
최종 판정
TSS 점수와 필터 결과를 종합해 매수 추천 / 보유 / 관망 / 매도 중 하나의 판정을 내립니다.
8
분석 리포트 생성
모든 분석 결과를 한눈에 볼 수 있는 리포트로 정리합니다. 왜 이 점수가 나왔는지 투명하게 공개합니다.
SCORING

5대 평가 카테고리

각 카테고리는 기본 50점에서 시작하여, 해당 지표의 상태에 따라 가감됩니다.

💰
가치 (Value)
  • PER: 낮을수록 가점 (8 미만 +20, 15 미만 +10)
  • PBR: 1 미만 +10, 5 초과 -10
  • 52주 위치: 저점 근처 +10, 고점 근처 -10
  • 목표가 괴리: 상승 여력 30% 이상 +15
📈
성장 (Growth)
  • 60일 수익률: 20% 이상 +20, 10% 이상 +15
  • EPS(주당순이익): 양수일 경우 +10
  • 하락률: -10% 이하 -15
⚡
모멘텀 (Momentum)
  • RSI: 60~75 강세 +15, 75 초과 과매수 -10
  • MACD 히스토그램: 양수 +10, 음수 -5
  • 볼린저밴드 %B: 30~70 안정 +5
🔄
수급 (Supply)
  • 거래량 비율: 평균 2배 + 상승 시 +20
  • 거래량 급증 + 하락: -15 (투매 징후)
  • 거래 부진 (평균 50% 미만): -5
📊
기술적 분석 (Technical)
  • 이동평균선 정배열: 5일선 > 20일선 > 60일선 > 120일선 위에 현재가 → 최대 +35
  • 골든크로스 경향 (5일선 > 20일선) +5 / 데드크로스 경향 -5
  • 변동성: 20% 미만 안정 +5, 50% 초과 고위험 -10

TSS 점수를 5단계 등급으로 변환합니다.

등급TSS 점수의미투자 판단

SAFETY

5대 안전 필터

TSS 점수만으로는 부족합니다. 아래 5가지 필터 중 2개 이상 경고가 뜨면 TSS와 무관하게 관망/회피 판정을 내립니다.

📉
급락 경고
최근 5일간 -10% 이상 하락하면 작동. 단기 낙폭이 과도한 상태에서 진입하는 위험을 방지합니다.
🔊
거래량 이상
거래량이 평균의 5배 이상 급증 + 하락 동시 발생 시 작동. 세력 매도 또는 투매 징후를 포착합니다.
💸
과대평가
PER 60배 초과 또는 PBR 10배 초과 시 작동. 실적 대비 주가가 비정상적으로 높은 상태를 경고합니다.
⚡
추세 이탈
현재가가 60일 이동평균선보다 10% 이상 아래에 있으면 작동. 중기 추세가 완전히 무너진 상태입니다.
🎢
변동성 과다
연간 변동성 60% 초과 시 작동. 하루 등락폭이 과도해 예측 불가능한 상태를 의미합니다.
DATA

데이터 출처

행운의서동이는 아래 데이터를 실시간으로 수집합니다. 수동 입력이 아닌 API 자동 수집입니다.

  • 한국 주식: 네이버 증권 API — 현재가, 전일대비, 시간외 가격, PER, PBR, EPS, 52주 고/저, 목표가, 1년 일봉 차트
  • 미국 주식: Yahoo Finance API — 현재가, 전일종가, 1년 일봉 차트, 52주 고/저
  • 갱신 주기: 분석 실행 시점의 최신 데이터 사용. TOP 20은 장중 10분, 장외 30분마다 자동 갱신
